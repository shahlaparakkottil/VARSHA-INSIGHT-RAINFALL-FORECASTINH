# Varsha Insight: Data-Driven Rainfall Monitoring and Forecasting

This project predicts future rainfall trends across India using machine learning and provides rainfall alerts and farmer advisory recommendations.

## Technologies Used
- Python
- XGBoost
- Random Forest
- Streamlit
- SQL
- R Programming
- Hadoop MapReduce
- Apache Spark
- Tableau
- Power BI

## Features
- Rainfall Forecasting (2018–2030)
- Drought and Heavy Rainfall Alerts
- Farmer Advisory System
- Interactive Dashboards
- Distributed Data Processing

## Folder Structure
- dashboards: Tableau and Power BI files
- dataset: Project datasets
- models: Streamlit, Hadoop, Spark, R, SQL files
- notebooks: Jupyter notebook
- reports: Documentation
